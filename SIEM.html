Deployed Splunk to implement a comprehensive security monitoring solution for a business, integrating logs from:

🌐 Routers & a network controller

🛡️ Endpoint Detection & Response (EDR) tools

⚠️ Intrusion Detection/Prevention System (IPS/IDS) tools

📡 Vulnerability scanners

The dashboard provides visibility into:

✅ Connected, failed, and disconnected devices
🔑 Authentication attempts with top failed devices (Possible Brute-force)
🖥️ Admin panel access activity (IP, MAC, timestamp)
🛡️ Malware detections from endpoints
🚨 Critical & high-severity alerts
📌 Devices generating the highest number of alerts (Possibly compromised)
📊 Vulnerability insights with CVSS scoring
🕵️‍♂️ Attack types and priority levels

Key skills applied & strengthened:

- Log ingestion from multiple data sources
- Parsing, correlation, and detection with SPL
- Building SIEM-style visualizations for threat monitoring

This project solidified my experience in designing security monitoring workflows and taught me how different data sources contribute to detecting, investigating, and responding to threats.
